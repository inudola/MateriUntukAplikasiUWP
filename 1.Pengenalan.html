Pengenalan HTML


HTML (Hypertext Markup language) adalah bahasa standard yang digunakan untuk menampilkan web. Yang bisa dilakukan dengan HTML yaitu:


  • 1. Mengatur tampilan dari halaman web dan isinya.
    • 2. Membuat table dalam halaman web.
  • 3. Mempublikasikan halaman web secara online.
  • 4. Membuat Form yang digunakan untuk menangani registrasi dan transaksi via web.
  • 5. Menambahkan objek-objekseperti citra, audio, video, animasi, java applet dalam halaman web.
  • 6. Menampilkan area gambar (Canvas) di browser.
