Head, Title, Body

Head

Elemen head kepala dari dokumen HTML, tag <head> dan </head> terletak di antara tag <html> dan </html>.


Title

Elemen title merupakan judul dari dokumen html yang ditampilkan pada judul jendela browser.


Body

Elemen ini untuk menampilkan isi dokumen html


Contoh

<HTML>

<head>

<title>Judul</title>

</head>

<body>

Elemen ini untuk menampilkan isi dokumen html

</body>

</HTML>
