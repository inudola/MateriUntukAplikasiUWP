ELEMEN HTML 5


Berikut adalah Tag baru yang terdapat pada dokumen HTML 5, antara lain:

TAG DESKRIPSI
<ARTICLE> Tag untuk mendefinisikan sebuah artikel
<ASIDE> Tag untuk mendefinisikan konten selain dari konten pada halaman.
<BDI> Tag yang digunakan untuk mengisolasi bagian dari teks yang dapat diformat dalam arah yang berbeda dari teks lain di luar itu.
<COMMAND> Tag untuk mendefinisikan tombol perintah bahwa seseorang pengguna dapat memanggil.
<DETAILS> Tag untuk mendefinisikan rincian tambahan bahwa pengguna dapat melihat atau menyembunyikan.
<SUMMARY> Tag untuk mendefinisikan sebuah ringkasan dari suatu elemen tag <DETAILS>.
<FIGURE> Tag yang digunakan untuk menentukan konten secara mandiri, seperti ilustrasi, diagram, foto, daftar kode, dan lain-lain.
<FIGCAPTION> Tag yang digunakan untuk mendefinisikan sebuah caption untuk elemen <FIGURE>.
<FOOTER> Tag untuk mendefinisikan sebuah bagian akhir.
<HEADER> Tag untuk mendefinisikan sebuah awalan pada dokumen.
<HGROUP> Group untuk menyetel heading dari <h1> sampai <h6>.
<MARK> Tag untuk menandai dan mendefinisikan suatu teks.
<METER> Tag untuk mendefinisikan pengukuran skalar dalam kisaran yang dikenal sebagai alat ukur.
<NAV> Tag untuk mendefinisikan navigasi link.
<PROGRESS> Tag untuk kemajuan suatu tugas.
<RUBY> Tag untuk mendefinisikan sebuah penjelasan teks ruby untuk tipografi asia timur.
<RT> Tag untuk mendefinisikan sebuah penjelasan atau pengucapan karakter. (Tipografi untuk asia timur).
<RP> Tag untuk mendefinisikan apa yang harus ditampilkan di browser yang tidak mendukung penjelasan bahasa ruby.
<SECTION> Tag untuk mendefinisikan bagian dalam dokumen.
<TIME> Tag untuk mendefinisikan tanggal atau waktu.
<WBR> Tag untuk mendefinisikan linebreak.
<CANVAS> Tag yang digunakan untuk menggambar grafik, On the fly, yang biasanya melalui script Javascript


Multimedia


Elemen multimedia baru pada HTML 5, antara lain:

TAG DESKRIPSI
<AUDIO> Tag untuk mendefinisikan isi suara.
<VIDEO> Tag untuk mendefinisikan sebuah video atau film.
<SOURCE> Tag untuk mendefinisikan beberapa sumber daya media untuk <VIDEO> dan <AUDIO>.
<EMBED> Tag untuk mendefinisikan sebuah wadah untuk aplikasi eksternal atau konten interaktif Plug-In
<TRACK> Tag yang digunakan untuk mendefinisikan trek teks untuk <VIDEO> dan <AUDIO>.


ELEMEN FORM


Berikut adadalah elemen Form dari HTML 5:

TAG DEKRIPSI
<DATALIST> Tag yang digunakan untuk menentukan daftar pilihan yang telah ditentukan untuk kontrol input.
<KEYGEN> Mendefinisikan sebuah bidang kunci-pair generator (Untuk formulir).
<OUTPUT> Mendefinisikan hasil perhitungan.


ATRIBUT INPUT


Berikut adadalah tipe atribut input

TIPE DEKRIPSI
COLOR Tipe input dari tipe warna hexadecimal, seperti #FFFFFF.
DATE Tipe input untuk tanggal.
DATETIME Tipe input untuk tanggal dan waktu.
DATETIME-LOCAL Tipe input untuk tanggal dan waktu lokal.
EMAIL Tipe input untuk satu atau lebih alamat email.
MONTH Tipe input untuk Bulan.
NUMBER Tipe input untuk angka.
RANGE Tipe input untuk jangkauan tertentu.
SEARCH Tipe input untuk Field pencarian.
TEL Tipe input untuk nomor telepon.
TIME Tipe input dari waktu.
URL Tipe input untuk Link atau URL.
WEEK Tipe input untuk minggu.
