TAG & elemen HTML


Penanda perintah di dalam HTML. Biasanya disebut dengan TAG digunakan untuk menentukan tampilan dari dokumen HTML.


Tag berpasangan:

HTML <BEGIN TAG>ISI DOKUMEN </END TAG>


Format Tag tidak berpasangan:

HTML <TAG>ISI DOKUMEN

Setiap dokumen HTML diawali dan diakhiri dengan TAG HTML:

HTML

<HTML>

<head>

</head>

<body>

</body>

</html>
