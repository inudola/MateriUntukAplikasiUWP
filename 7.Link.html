LINK


Link merupakan penghubung antara satu halaman dengan halaman lainnya.

Link merupakan suatu gambar atau teks dengan suatu alamat tertentu.


Contoh:


Gambar


Untuk menggunakan gambar menggunakan TAG <IMG>

HTML mendukungbanyak format gambar seperti GIF, JPEG, PX, PNG, WMF, DLL


Tag <IMG> memiliki atribut:


Atribut Keterangan
SRC Menunjukan URL atau direktori tempat file gambar berada**
WIDTH Lebar gambar dalam Pixel
HEIGHT Tinggi gambar dalam Pixel
ALIGN Posisi teks di sekitar gambar. Nilainya adalah TOP, MIDDLE, BOTTOM, LEFT, RIGHT. (Atribut ini tidak didukung di HTML 5)
ALT Menampilkan teks pengganti gambar jika gambar tidak ditampilkan

Contoh:

Coba tambahkan atribut pada kode di atas

** Gunakan ms-appx-web:///html/image/ sebagai direktori file berada, jika ingin mencoba pada aplikasi ini.
