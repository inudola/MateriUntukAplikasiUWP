Apa itu HTML 5?


HTML (HyperText Markup Language) 5 adalah sebuah bahasa markup untuk menstrukturkan dan menampilkan isi dari World Wide Web, sebuah teknologi ini dari internet. HTML 5 adalah versi kelima dari HTML.


HTML5 merupakan salah satu karya World Wide Web Consortium untuk mendefinisikan markah tunggal yang ditulis dengan cara HTML ataupun XHTML.


Meskipun HTML 5 dikenal luas oleh para pengembang web sejak lama, HTML 5 baru mencuat pada April 2010 setelah mendiang Apple Inc., Steve Jobs, mengatakan bahwa dengan pengembangan HTML 5, "Adobe Flash tidak dibutuhkan lagi untuk menyaksikan video atau menyaksikan konten apapun di web."
