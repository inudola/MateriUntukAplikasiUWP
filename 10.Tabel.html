TABEL


Tabel sering digunakan dalam website, misalnya mendampilkan suatu data seperti data siswa atau rekap keuangan. Dengan tabel biasanya ampilan akan terlihat rapi. Tampilan data dari database pun akan terlihat lebih rapi.

HTML menyediakan tag-tag untuk membuat sebuah tabel, yaitu:


TAG <TABEL>: Untuk mendefinisikan sebuah tabel.

TAG <TR>: Untuk mendefinisikan baris tabel.

TAG <TH>: Untuk mendefinisikan judul pada sel dalam sebuah tabel.

TAG <TD>: Untuk mendefinisikan isi tiap sel dalam tabel.


Atribut untuk tag <tabel> adalah sebagai berikut:

Atribut Keterangan
ALIGN Posisi horizontal tabel (Center, Justify, left, right)
BACKGROUND Menentukan gambar latar belakang tabel.
BGCOLOR Menentukan warna latar blakang tabel.
BORDER Menentukan tebal bingkai tabel.
BORDERCOLOR Menentukan wara bingkai tabel.
BORDERCOLORLIGHT Menentukan warna depan bingkai tabel.
BORDERCOLORDARK Menentukan warna bayangan bingkai.
CELLSPACING Menentukan jarak spasi antar sel.
CELLPADDING Menentukan jarak isi sel dengan bingkai.
HEIGHT Menentukan tinggi tabel.
WIDTH Menentukan lebar tabel.
VALIGN Menentukan posisi vertikal teks dalam sel. (BASELINE, TOP, BOTTOM, MIDDLE).


Atribut untuk tag <TD> adalah sebagai berikut:

Atribut Keterangan
ALIGN Posisi horizontal text dalamtabel (Center, Justify, left, right)
BACKGROUND Menentukan gambar latar belakang sel.
BGCOLOR Menentukan warna latar blakang sel.
BORDER Menentukan tebal bingkai sel.
BORDERCOLOR Menentukan wara bingkai sel.
BORDERCOLORLIGHT Menentukan warna depan bingkai sel.
BORDERCOLORDARK Menentukan warna bayangan sel.
COLSPAN Menentukan jumlah kolom yang digabung.
HEIGHT Menentukan tinggi sel.
NOWRAP Menentukan text agar tetap satu baris.
ROWSPAN Menentukan jumlah baris yang digabung.
VALIGN Menentukan posisi vertikal teks dalam sel. (BASELINE, TOP, BOTTOM, MIDDLE).
